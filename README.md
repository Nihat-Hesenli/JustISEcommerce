# 🛒 React & Redux E-Commerce Project

🔗 **Canlı Demo:** :https://just-is-ecommerce.vercel.app/

Bu layihə, modern frontend texnologiyalarından istifadə edərək hazırlanmış, sürətli və interaktiv bir alış-veriş platformasıdır. Layihə çərçivəsində dövlət idarəetməsi (state management), axtarış sistemləri və istifadəçi təcrübəsi (UX) fokus nöqtələri olaraq seçilmişdir.

## 🚀 Texnologiyalar
* **React.js** - UI komponentləri üçün
* **Redux Toolkit** - Qlobal səbət idarəetməsi üçün
* **Tailwind CSS** - Modern və responsiv dizayn üçün
* **React Icons** - Vizual ikonlar üçün
* **JSON Data** - Məhsul məlumatlarının idarə olunması üçün

## ✨ Əsas Funksiyalar

### 1. Səbət İdarəetməsi (Redux)
- **Dinamik Səbət:** Məhsulların əlavə edilməsi, silinməsi və sayının tənzimlənməsi.
- **Real-time Hesablama:** Hər məhsulun öz alt cəmi və səbətin ümumi məbləği (Total Price) avtomatik hesablanır.
- **Cart Drawer:** İstifadəçini yormayan, sağ tərəfdən açılan modern səbət paneli.

### 2. Axtarış və Sıralama Sistemi
- **Live Search:** Məhsul adlarına görə dərhal (real-time) filterləmə.
- **Price Sorting:** Məhsulların qiymətə görə "Ucuzdan bahaya" və "Bahadan ucuza" sıralanması.

### 3. İstifadəçi Təcrübəsi (UX)
- **Loading State:** Məlumatlar yüklənən zaman istifadəçiyə göstərilən animasiyalı spinner.
- **Empty State:** Səbət boş olduqda istifadəçini məlumatlandıran xüsusi interfeys.
- **Responsive:** Bütün ekran ölçülərinə (Mobil, Planşet, Desktop) tam uyğunluq.
